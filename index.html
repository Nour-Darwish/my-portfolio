AUBify is a platform developed by a team of four to enhance communication and engagement within the AUB community. It provides a safe space for students to connect, share information, and stay updated on university events and news, fostering a sense of belonging and unity.

MealMission

MealMission offers an innovative platform designed to connect potential food donors with receivers, providing a streamlined and efficient solution to tackle the issues of food waste and hunger. At its core, MealMission facilitates the donation process, ensuring that surplus food reaches those who need it most while minimizing waste.

MiddleEast Traveling System(METS)

This project focused on creating a database for a Middle East Traveling System to streamline travel planning and management. The system helps tourists explore various destinations, book trips, and engage in activities tailored to their interests. It also supports guides in organizing trips and provides detailed information about countries, cities, local events, and expected costs. Users can access trip schedules, activity details, and reviews, making it easier to plan enjoyable and well-informed travel experiences

Maze Game

This C++ program simulates a maze game where mazes are dynamically generated using Prim’s and paths are found using the A* search algorithm. Users can input maze dimensions and start/end coordinates to find a path through the maze, with the program outputting whether a path was found or not

Medical language model

Developed a prototype for a domain-specific language model by fine-tuning the Meta Llama 2 7B foundation model using Amazon SageMaker and other AWS tools. The model was trained on a copyright-free medical dataset to generate accurate and contextually relevant responses, enabling applications such as medical chatbots, internal knowledge systems, and automated content creation. This project helped me improve my skills in machine learning and natural language processing while gaining practical experience with AWS tools to address challenges in the healthcare domain.
